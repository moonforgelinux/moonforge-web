# Moonforge Website

### Setting up

#### System Dependencies

Fedora:

```sh
$ sudo dnf install hugo npm
```

Node.js:

```sh
$ npm install autoprefixer postcss postcss-cli
```

#### Clone

```sh
$ git clone https://github.com/moonforgelinux/moonforgelinux.github.io.git
```

#### Initialize

```sh
$ cd moonforgelinux.github.io
$ hugo serve
```
